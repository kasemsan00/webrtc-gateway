Admin Web UI สำหรับจัดการระบบ SMS Link 

1. Frame สำหรับพัฒนา React Refine  
2. เว็บไซต์จะอยู่ในรูปแบบ Container File  
3. เว็บ Admin WebUI สามารถรองรับการ Single SignOn ผ่าน KeyCloak  
4. หน้า Dashboard สำหรับ Admin UI  
   1. ฟังก์ชันดูสรุปสถานะระบบ (จำนวน log, จำนวนห้องสนทนาที่เปิดอยู่, cron status, health check ของแต่ละบริการ)  
   2. ฟังก์ชันดูสถานะ cron job และรายการกิจกรรมล่าสุด (Log)  
   3. ฟังก์ชันรีเฟรชข้อมูล Dashboard ทุก 30 วินาที  
5. Admin  Web UI ส่ง SMS Link   
   1. ฟังก์ชันส่ง SMS วิดีโอและพิกัดสำหรับ RNIS จากเบอร์มือถือ  
   2. ฟังก์ชันเปิด tab browser อัตโนมัติสำหรับเจ้าหน้าที่ Admin SMS Video เมื่อส่ง SMS สำเร็จ  
   3. ฟังก์ชันเลือกภาษาต้นทางและปลายทางสำหรับแปลภาษาได้แก่ อังกฤษ (English), จีน (Chinese), เกาหลี (Korean), ญี่ปุ่น (Japanese), รัสเซีย (Russian), ฮินดี (Hindi), เยอรมัน (German), ฝรั่งเศส (French)  
   4. ฟังก์ชันแสดงพิกัดของผู้ใช้งานผ่าน WebUI  
   5. ฟังก์ชันแสดงสถานะการส่ง SMS สร้างลิงก์, เปิดลิงก์, ส่งพิกัด, วิดีโอคอล เป็นต้น  
   6. ฟังก์ชันแสดงประวัติการส่ง SMS ประกอบด้วย หมายเลขเคส, เบอร์โทรศัพท์, วัน-เวลา, สถานะ และศูนย์ปฏิบัติการ  
   7. ฟังก์ชันแสดงวิดีโอที่ทำการบันทึก  
6. Admin Web UI แสดง Log การใช้งานและสถิติ  
   1. ฟังก์ชันดูตารางรายการส่งลิงก์/ส่งข้อความของวิดีโอและพิกัด  
   2. ฟังก์ชันค้นหาและกรองข้อมูลสถิติ (search, linkType, ช่วงวันที่, sort)   
   3. ฟังก์ชันรีเฟรชข้อมูลหน้าสถิติ  
   4. ฟังก์ชันแสดงสถิติการใช้งาน  
   5. สามารถเลือกช่วงวันที่ และ Apply/Reset  
   6. สามารถดูแยกตามประเภทวิดีโอและพิกัด  
   7. สามารถดูแยกตามจังหวัดหรือศูนย์ได้  
   8. สามารถดูแยกตามอุปกรณ์ได้ (Android, iOS, PC)  
   9. กราฟแนวโน้มรายวัน และกราฟแยกตามอุปกรณ์/ประเภทลิงก์  
7. Admin Web UI ตั้งค่าระบบ  
   1. ฟังก์ชันการตั้งค่า SMS Domain ของวิดีโอและพิกัด  
   2. ฟังก์ชันการตั้งค่า SMS Header   
   3. ฟังก์ชันการตั้งค่าข้อความ Prefix ของ SMS   
   4. ฟังก์ชันตั้งค่า Node ของ LiveKit

 API กลางระบบ SMS Link

1. ฟังก์ชันรับคำสั่งเริ่มการแปลภาษา โดยรับข้อมูลแปรภาษาและชื่อห้องสนทนาและค่าการตั้งค่าภาษาต้นทางและปลายทางจาก Web UI  
2. ฟังก์ชันตรวจสอบความพร้อมของ Worker แปลภาษา  
3. ฟังก์ชันสร้างบอทแปลภาษาของ LiveKit (Translator-Bot) ไปยังห้องสนทนาและเริ่มคำสั่งแปลภาษา  
4. ฟังก์ชั่นอัพเดทสถานะการทำงาน (status) แปลภาษา  
5. ฟังก์ชันหยุดการแปลภาษาและอัพเดทสถานะการแปลภาษา  
6. ปรับเปลี่ยนการใช้ raw mysql มาเป็น prisma orm  
7. ปรับปรุงฐานข้อมูลให้รองรับการแปลภาษาและสถานะการแปลภาษา  
8. ฟังก์ชันสำหรับตรวจสอบสถานะการแปลภาษาแบบละเอียด โดยสามารถดูสถานะ session ของห้องสนทนาได้

API Translate Worker แปลภาษา

1. ฟังก์ชันสำหรับตรวจสอบ HealCheck ของ Worker แปลภาษา  
2. ฟังก์ชันสำหรับตรวจสอบสถานะความพร้อมของ API แปลภาษาและ API ปลายทาง  
3. ฟังก์ชันตรวจสอบสถานะความพร้อมของ API แปลภาษาปลายทาง   
4. ฟังก์ชันเริ่ม Session สำหรับแปลภาษาตามห้องสนทนาตามพารามิเตอร์ role, source language, target language และ TTS voice  
5. ฟังก์ชันสำหรับตรวจสอบสถานะ Session แปลภาษา  
6. ฟังก์ชันสำหรับปิด Session แปลภาษาของห้องสนทนา  
7. ฟังก์ชันสำหรับตรวจสอบสถานะการทำงานแปลภาษา  
8. ฟังก์ชันรองรับการส่งข้อมูลแปลภาษาผ่าน Data Message Socket

Web UI สำหรับวิดีโอคอลและพิกัด

1. เพิ่มปุ่มแปลภาษาและแสดงเฉพาะเจ้าหน้าที่สำหรับเริ่มแปลภาษา  
2. ฟังก์ชันเรียก API กลางสำหรับเริ่มแปลภาษาและหยุดการแปลภาษา  
3. ฟังก์ชันแปลภาษาข้อความสนทนาของเจ้าหน้าที่และผู้ใช้งาน  
4. ฟังก์ชันแสดง subtitle overlay การแปลภาษาทั้งฝั่งเจ้าหน้าที่และผู้ใช้งานวิดีโอคอล  
5. ฟังก์ชัน  subscribe translated เสียงแปลภาษาตามประเภทเจ้าหน้าที่และผู้ใช้งาน  
6. ฟังก์ชันแสดงตัวเลือกภาษาสำหรับผู้ใช้งานต่างประเทศได้แก่ อังกฤษ (English), จีน (Chinese), เกาหลี (Korean), ญี่ปุ่น (Japanese), รัสเซีย (Russian), ฮินดี (Hindi), เยอรมัน (German), ฝรั่งเศส (French)  
7. ปรังปรุง Web UI ให้รองรับเมนูภาษา อังกฤษ (English), จีน (Chinese), เกาหลี (Korean), ญี่ปุ่น (Japanese), รัสเซีย (Russian), ฮินดี (Hindi), เยอรมัน (German), ฝรั่งเศส (French)
